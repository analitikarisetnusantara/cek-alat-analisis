Mengapa uji asumsi wajib dilakukan?
Alat statistik parametrik dibangun di atas asumsi matematis. Melanggar asumsi ini membuat hasil analisis tidak valid, sekeras apapun kamu berusaha menginterpretasikannya.
Uji Normalitas
Apakah distribusi data membentuk kurva normal (bell-shaped)?
Kapan Digunakan
Sebelum menggunakan t-test, ANOVA, Pearson, atau Regresi Linear. Wajib dilakukan pada variabel dependen (dan residual untuk regresi).
Alat Uji
Shapiro-Wilk — untuk n < 50 (lebih akurat untuk sampel kecil)
Kolmogorov-Smirnov — untuk n ≥ 50
Q-Q Plot — inspeksi visual tambahan
Interpretasi
p > 0.05 → data normal → lanjutkan parametrik
p < 0.05 → data tidak normal → gunakan non-parametrik atau transformasi data
Solusi Jika Tidak Normal
Transformasi: log(x), √x, 1/x, Box-Cox
Atau langsung pakai alternatif non-parametrik (Spearman, Mann-Whitney, Kruskal-Wallis)
Uji Homogenitas Varians (Levene's Test)
Apakah varians antar kelompok bersifat homogen?
Kapan Digunakan
Sebelum t-test independen dan ANOVA. Membandingkan varians antar kelompok.
Alat Uji
Levene's Test — paling umum di SPSS
Bartlett's Test — untuk data normal sempurna
Interpretasi
p > 0.05 → varians homogen → lanjutkan t-test / ANOVA biasa
p < 0.05 → tidak homogen → gunakan Welch's t-test atau Games-Howell
Di SPSS
Analyze → Compare Means → Independent Samples T-Test → cek kolom "Levene's Test" di output
Uji Multikolinearitas (VIF)
Apakah variabel independen tidak saling berkorelasi terlalu tinggi?
Kapan Digunakan
Khusus untuk Regresi Linear Berganda (≥2 variabel independen). Tidak diperlukan untuk regresi sederhana.
Alat Ukur
VIF (Variance Inflation Factor): VIF < 10 = tidak ada multikolinearitas
Tolerance: > 0.1 = aman
Interpretasi
VIF < 5 → aman (ideal)
VIF 5–10 → perhatian
VIF > 10 → multikolinearitas serius → hapus atau gabung variabel
Di SPSS
Analyze → Regression → Linear → Statistics → centang Collinearity diagnostics
Uji Heteroskedastisitas
Apakah varians residual bersifat konstan (homoskedastik)?
Kapan Digunakan
Khusus untuk analisis Regresi Linear. Residual harus bervarians konstan.
Alat Uji
Scatterplot residual — inspeksi visual
Glejser Test — formal, p > 0.05 = homoskedastik
Breusch-Pagan — di R/EViews
Interpretasi Plot
Titik tersebar acak tanpa pola → OK (homoskedastik)
Ada pola corong/kipas → heteroskedastisitas → masalah
Solusi
Transformasi variabel (log Y), gunakan robust standard errors, atau Weighted Least Squares (WLS)
Uji Autokorelasi (Durbin-Watson)
Apakah residual antar observasi tidak saling berkorelasi?
Kapan Digunakan
Regresi Linear, terutama data time series atau data panel. Jarang kritis untuk data cross-sectional murni.
Nilai DW
DW ≈ 2 → tidak ada autokorelasi (ideal)
DW < 1.5 → autokorelasi positif
DW > 2.5 → autokorelasi negatif
Solusi
Tambahkan lag variabel, gunakan Cochrane-Orcutt, atau pindah ke model ARIMA/panel data.
Validitas & Reliabilitas Instrumen harus diuji sebelum pengumpulan data utama, bukan setelah. Idealnya dilakukan pada uji coba (pilot study) dengan 30 responden.
Uji Validitas
Apakah instrumen mengukur apa yang seharusnya diukur?
Validitas Isi (Content Validity)
Penilaian ahli (expert judgment) terhadap relevansi item dengan konstruk. Gunakan CVR (Content Validity Ratio) atau CVI.
Validitas Konstruk
Exploratory Factor Analysis (EFA) — untuk instrumen baru
Confirmatory Factor Analysis (CFA) — untuk instrumen yang sudah ada teorinya
Item valid jika loading factor ≥ 0.5
Validitas Kriteria (Pearson)
Korelasi skor item dengan skor total. Item valid jika r-hitung > r-tabel (biasanya > 0.30 untuk n=30).
AVE (untuk PLS-SEM)
Average Variance Extracted > 0.5 = validitas konvergen terpenuhi. Discriminant validity: √AVE > korelasi antar konstruk.
Uji Reliabilitas
Apakah instrumen menghasilkan pengukuran yang konsisten?
Cronbach Alpha (α)
Paling umum digunakan. α ≥ 0.70 = reliabel. α ≥ 0.80 = sangat reliabel. α < 0.60 = perlu revisi item.
Composite Reliability (CR)
Digunakan dalam PLS-SEM. CR > 0.70 = reliabel. Lebih robust dari Alpha karena mempertimbangkan loading.
Split-Half / Test-Retest
Split-half: bagi item jadi dua bagian, korelasi keduanya. Test-retest: ukur dua kali, korelasi hasilnya > 0.70.
Di SPSS
Analyze → Scale → Reliability Analysis → pilih Alpha → masukkan semua item satu konstruk.

Triangulasi — Strategi Kredibilitas Utama
Menggunakan lebih dari satu sumber/metode untuk mengkonfirmasi temuan
Triangulasi Sumber
Data dikumpulkan dari berbagai sumber berbeda (manajer, karyawan, dokumen) untuk mengonfirmasi konsistensi temuan.
Triangulasi Metode
Kombinasi wawancara + observasi + dokumen. Jika ketiganya mengarah ke temuan yang sama = kredibilitas tinggi.
Triangulasi Teori
Interpretasi data dengan beberapa lensa teori berbeda untuk melihat apakah hasilnya konsisten.
Member Checking
Kembalikan hasil analisis ke partisipan untuk dikonfirmasi apakah interpretasi peneliti sudah tepat dan akurat.

Kapan Mixed Methods?Panduan
Mixed Methods bukan sekadar menggabungkan kuesioner dan wawancara. Ini adalah desain penelitian yang terstruktur dengan logika integrasi yang jelas antara data kuantitatif dan kualitatif (Creswell & Plano Clark, 2018).
1
Filter M1 — Urutan Pengumpulan Data
Bagaimana urutan pengumpulan data kuantitatif dan kualitatif dalam penelitianmu?
2
Filter M2 — Prioritas / Bobot
Komponen mana yang menjadi inti penelitianmu?
3
Filter M3 — Tujuan Integrasi
Apa tujuan menggabungkan kedua jenis data tersebut?
Rekomendasi Desain Mixed Methods
⬆

Pilih ketiga filter untuk melihat rekomendasi desain Mixed Methods.

Gunakan Mixed Methods JIKA…
  • ✅ Pertanyaan penelitian membutuhkan keduanya — Angka menunjukkan BERAPA BESAR masalahnya, narasi menjelaskan MENGAPA masalah itu terjadi.
  • ✅ Hasil kuantitatif perlu penjelasan lebih dalam — Statistik menunjukkan korelasi signifikan, tapi mengapa? Wawancara menjawabnya.
  • ✅ Membangun instrumen dari eksplorasi awal — Kualitatif dulu untuk memahami konstruk, lalu buat kuesioner berdasarkan temuan.
  • ✅ Triangulasi untuk validasi silang — Membuktikan temuan dari dua arah berbeda untuk meningkatkan kepercayaan.
JANGAN Gunakan Mixed Methods JIKA…
  • ❌ Sekadar menambahkan wawancara agar "terlihat lengkap" — Mixed methods bukan pelengkap. Integrasi harus bermakna dan direncanakan sejak awal.
  • ❌ Satu metode sudah cukup menjawab pertanyaan — Jika kuesioner saja sudah menjawab tujuanmu, tambahan wawancara hanya membuang waktu.
  • ❌ Sumber daya tidak memadai — Mixed methods membutuhkan dua kali upaya. Jika waktu dan akses terbatas, pilih satu paradigma dan lakukan dengan sangat baik.
Tantangan Utama Mixed Methods di Skripsi
Waktu & Sumber Daya
Harus menguasai dua paradigma sekaligus. Waktu pengumpulan dan analisis data berlipat ganda dibanding satu paradigma.
Integrasi yang Bermakna
Kesalahan terbesar: menulis bab kuantitatif dan bab kualitatif secara terpisah tanpa benar-benar mengintegrasikan keduanya di pembahasan.
Kompetensi Ganda
Harus bisa membaca output SPSS/SmartPLS sekaligus melakukan coding tematik di NVivo. Konsultasi intensif dengan pembimbing sangat disarankan.
Jumlah Sampel Berbeda
Sampel kuantitatif (besar, acak) berbeda dari sampel kualitatif (kecil, purposive). Keduanya bisa berasal dari populasi yang sama atau berbeda.
